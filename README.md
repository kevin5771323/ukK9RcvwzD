# 停车场管理系统 java1146

本项目是一款基于前后端分离架构的停车场管理系统，采用Java和Vue技术栈进行开发。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows
- 开发环境：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 8.0
- 技术框架：Java、Mysql、SpringBoot、Mybatis、Vue

## 功能模块

### 系统角色

#### 管理员角色

- 首页：返回系统首页。
- 用户信息管理：管理停车场相关用户的信息。
- 车位信息管理：对停车场车位信息进行管理，包括车位编号和位置等。
- 车位费用设置：设置车位收费标准等费用相关内容。
- 停泊车辆查询：查询当前在停车场停泊的车辆信息。
- 车辆进出管理：处理车辆进出停车场的操作与记录。
- 登录日志查询：查看系统用户的登录日志信息。

#### 用户角色

- 首页：返回系统起始界面。
- 个人中心：查看和管理用户基本信息、设置等。
- 预定停车位：用户进行停车位预定操作。
- 缴费信息：管理用户的缴费记录。

## 运行环境

- 前端：现代浏览器（建议使用Chrome或Firefox）
- 后端：JDK 1.8及以上版本的Java运行环境

1. 前后端分离设计，便于维护和扩展。
2. 采用SpringBoot框架，提高开发效率。
3. 集成Mybatis，简化数据库操作。
